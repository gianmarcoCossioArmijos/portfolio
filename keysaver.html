Key Saver

Ver en GitHub

Mini sistema web que genera contraseñas seguras y las administra. Este proyecto contiene metodos crud para guardar, editar, eliminar, y mostrar o reportar las contraseñas pertenecientes a un usuario, ademas tambien contiene un metodo para crear y editar usuarios. En el Backend se ha usado Servlets, y MySQL, y en el Frontend JSP y CSS.

Tecnologias

  • Servlets
  • JSP
  • JSTL
  • MySQL
